发表于 2023-04-19 | 更新于 2023-04-14 | 条评论
欢迎来到缘的博客，接下来由我来为大家介绍在安装和使用后出现解决命令提示符(cmd)中输入 (net start mysql)提示服务名无效或者MySQL正在启动或MySQL无法启动的问题。 一 命令行工具的界面上输入 (net start mysql)提示服务名无效。 1-出现以下的情况时，因为net start +服务名,启动的是windows下注册的服务，当前路径下没有mysql服务。 2-解决办法：来到(每个人的安装路径不同)MySQL的安装路径下bin。 3-在命令行中输入mysqld --install，此时已经启动成功。 4-如果不成功，出现以下情况，则需要通过管理员身份运行命令提示符。 二 执行net start mysql出现 则需要删除mysql下的data文件，重新执行 mysqld --initialize 就可以在当前路径下生成data文件夹，再执行net start mysql 就可以启动mysql。 三 将小皮面板关闭就可以解决问题了。
